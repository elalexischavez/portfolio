This proyect is a mobile application maked with react native and android studio, it allows you use a mobile device to send and resive data from mobile device to a specific hardware using arduino and HC06 bluetooth module. The purpose of this proyect was to understand how wireless connections works and relate it with all osi model layers.

ENCLICLOVERSE

This application consist in a cosmos library where you can search for information about the objects on the infiinite cosmos and watch information about them. Also you can share with other users anything you want through the chat. In addition is implemented a mongodb autocomplete search engine.
